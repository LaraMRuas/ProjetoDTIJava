Decisões de Proejto:

- Front-End: Feito em React Native, interface responsiva, laytou simples e intuitivo somente com as funcinoalidades requeridas
- Back-End: Feito em Java com os modelos (Aluno, Aluno Processado, Relatorio e Relatorio Requerido). Utilizando a arquitetura MVC. Back-end utiliza o framework springboot. Back-end é uma API RestFull.



Lista de premissas assumidas
 - É uma aplicação que não utiliza banco de dados
 - A lista não contem as funções de edição e deleção
